# 作用

- 去除字符串两端的空格

# 语法

- 字符串.trim（）

- 相当于字符串都会自带这个方法

# 例子

~~~JavaScript
<script>
        //相当于字符串都会自带这个方法
        const str = '     Hatti     '
        //字符串.trim（）表示，字符串去除左右两边空格剩下的
        console.log(str.trim());
</script>
~~~
